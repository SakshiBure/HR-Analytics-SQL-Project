# 📊 HR Analytics SQL Project

## 🚀 Project Overview
This project analyzes employee data using SQL across multiple HR tables.

## 🗄️ Tables Used
- Employee
- Department
- EmployeePayHistory
- EmployeeDepartmentHistory
- Shift

## 🧠 Key Analysis
- Employee full profile
- Department-wise employee count
- Salary analysis
- Current employee tracking

## ⚙️ Skills Used
- SQL Joins
- Aggregations
- Views
- Data Analysis

## 📈 Future Scope
- Power BI Dashboard Integration
- KPI Tracking
